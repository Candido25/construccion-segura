Errores en construcción

Mezcla incorrecta del concreto: un error silencioso que baja la calidad de toda la obra.

En muchas obras pequeñas se mezcla al ojo. El problema es que el concreto no perdona improvisaciones. Si cambias las proporciones de cemento, arena, piedra y agua sin control, la resistencia final puede ser menor a la que necesita tu vivienda.

Error común

Preparar el concreto sin dosificación clara, usando más arena de la debida, menos cemento o cambiando cantidades según lo que se vea bien en el momento.

Por qué ocurre

Porque se busca rapidez, se confía solo en la costumbre o no se entiende que cada elemento tiene una resistencia esperada. También influye no medir con baldes, latas o proporciones verificables.

Consecuencias

Un concreto mal mezclado puede fisurarse más, resistir menos y envejecer peor. El elemento puede verse terminado, pero no necesariamente está cumpliendo lo que debería soportar.

Cómo hacerlo correctamente

Usa una dosificación definida para el tipo de elemento y mide los materiales de forma constante. Si una mezcla funcionó para un sobrecimiento, no significa que sirva igual para una columna o viga.

Advertencia

Un concreto pobre puede dejar problemas que no se ven el mismo día.

El riesgo de una mala mezcla no siempre se nota durante el vaciado. A veces aparece después como fisuras, desprendimientos o menor desempeño estructural.

Solución técnica explicada fácil

Define la proporción antes de empezar

No improvises mientras mezclas. Toda la cuadrilla debería manejar la misma relación de materiales.

Mide siempre igual

Si hoy usas una lata y mañana una carretilla, ya cambiaste la mezcla. La consistencia es clave para construir correctamente.

Distingue el uso del elemento

No todos los concretos trabajan igual. Una columna necesita más cuidado que un piso simple.

Checklist rápido antes de avanzar

Antes de seguir

  • Define dosificación y responsable.
  • Mide materiales con recipientes constantes.
  • Controla que no cambien proporciones en plena obra.

Durante la ejecución

  • Evita lotes muy secos o demasiado fluidos.
  • No agregues agua por costumbre.
  • Revisa homogeneidad antes del vaciado.

Mensaje clave

Evita este error antes de construir. El concreto no debería prepararse como salga, sino como el elemento lo necesita.
